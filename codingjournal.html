Let's learn about command lines and how to use them in your terminal


Command Lines Definition
pwd Print current directory
cd Change the current directory
ls List everything in that (current) directory
mkdir Make a directory
touch Make a file
rm -r Delete a directory


'Terminal' is basically where you can write command lines for the computer to do whatever you want it to do!
  1. First, find a big search icon üîç in the top right-hand corner of your computer
  2. Type in 'Terminal'
  3. Into the terminal, type in 'pwd'
  4. It will tell you where you currently are (what folder you are in)
  5. You can type 'cd desktop' to change the repository to desktop
  6. Then type 'mkdir (whatever name you like)' to make a new folder
  7. Once cd'ing into the new directory/folder on your desktop, type 'touch index.html'
  8. Now, you can type 'ls' to check if the file has been made in the respository
